Follow these steps to include MDI to your Angular Material project. (This asumes that you already know how to create an Angular app using the CLI and you know the basics of Angular.)

  1. Save mdi.svg to your assets folder. (located in src/assets) Alternatively, you can get mdi.svg from this website by using https://chan4077.github.io/res/mdi.svg.

  2. In your app's module, import MatIconModule and MatIconRegistry from @angular/material as well as HttpModule from @angular/http and DomSanitizer from @angular/platform-browser. Place MatIconModule under imports.

    import { MatIconModule, MatIconRegistry } from '@angular/material'; import { HttpModule } from '@angular/http'; @NgModule({ imports: [ HttpModule, MatIconModule ] })

    If you're using version 5.0.0-rc0, use this code instead:

    import { MatIconModule, MatIconRegistry } from '@angular/material'; import { HttpClientModule } from '@angular/common/http'; @NgModule({ imports: [ HttpClientModule, MatIconModule ] })

  3. Next, add a constructor to your app's module as below:

    export class AppModule { constructor(private matIconRegistry: MatIconRegistry, private dom: DomSanitizer){ matIconRegistry.addSvgIconSet(dom.bypassSecurityResourceUrl('./assets/mdi.svg')); } }

  4. After that, add the following classes to your styles.css (or styles.scss, or wherever your app-specific styles are):

    button.mat-menu-item { line-height: 24px !important; } a.mat-menu-item > mat-icon { margin-bottom: 14px; } .mat-icon svg { height: 24px; width: 24px; }

  5. Finally, to use these icons, use the svgIcon attribute of mat-icon:

    <mat-icon svgIcon="account"></mat-icon>

    And you're done!
